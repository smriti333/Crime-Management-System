# Crime Management System

## Description
A database system designed to manage crime records efficiently using SQL.

## Features
- Store crime records
- Maintain offender information
- Manage location-based crime data
- Supports relational database structure with primary and foreign keys

## Tools Used
- SQL / PL-SQL
- Oracle Database
